This is a basic Employee Management Record

Add: Add elements (no field should be empty)

Delete: Delete elements (based on Employee ID or Name, filling any other field would give error)

Update: Update the values of table (based only on Employee ID, use must enter their Employee ID and then whichever other field has a value, that value will be updated otherwise remain unchanged)

Search: Search the values of table (based on Employee ID or Name, filling any other field would give error)

First / Last: Returns the value of the first / last field entered

Previous / Next: Returns the value of the previous / next to the field selected or currently being displayed

Show All: Displays the table
